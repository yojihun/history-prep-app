# 📊 데이터 스키마 및 타 과목 설정 템플릿 (Data Schema Guide)

모든 학습 자료와 평가 문항은 `src/data/chapters.js` 한 곳에서 통합 관리됩니다. 구조를 올바르게 준수해야 빈칸 학습 및 오디오 기능이 충돌 없이 작동합니다.

---

## 1. 데이터 전체 구조 명세 (JSON Schema)

`chapters` 배열 내부의 각 단원 개체는 다음과 같은 키 구조를 지닙니다:

```json
{
  "id": "단원ID (문자열)",
  "title": "대단원/소단원 대제목 (문자열)",
  "sections": [
    {
      "id": "섹션ID (예: 1-1)",
      "title": "중단원 제목",
      "subsections": [
        {
          "id": "소단원ID (예: 1-1-1)",
          "title": "소단원 세부 소제목",
          "objective": "★ 학습 목표 내용 (AI가 학습목표 서술을 검사할 때 채점 기준 문장으로 활용됨)",
          "page": "교과서 페이지 번호 (숫자)",
          "imageUrl": "선택사항: 설명 이미지 경로 (예: '/images/pottery.png')",
          "imageCaption": "선택사항: 이미지의 하단 캡션 설명",
          "fillInTheBlanks": [
            {
              "sentence": "대괄호 []를 넣은 빈칸용 문제 문장 (예: '물은 [수소]와 산소로 결합되어 있다.')",
              "answer": "정답 단어 (쉼표 구분)"
            }
          ],
          "flashcards": [
            {
              "term": "단원 핵심 용어 명칭",
              "definition": "용어에 대한 상세한 개념 설명 문장 (TTS 읽기에 포함됨)"
            }
          ]
        }
      ]
    }
  ],
  "subjectiveQuizzes": [
    {
      "id": "서술형 문제ID (예: subq_1_1)",
      "question": "서술형 문항 질문 내용",
      "expectedAnswer": "AI 채점 시 비교 기준이 되는 상세 모범 답안 / 필수 핵심 키워드 목록"
    }
  ]
}
```

그리고 실전 객관식 퀴즈는 파일 하단에 `quizzes` 상수로 분리해서 배치합니다:

```javascript
export const quizzes = [
  {
    "id": "객관식퀴즈ID (숫자)",
    "chapterId": "연결할 단원ID (예: '1')",
    "question": "4지선다형 질문",
    "options": [
      "선택지 1",
      "선택지 2",
      "선택지 3",
      "선택지 4"
    ],
    "answer": "정답 인덱스 (0, 1, 2, 3 중 하나)",
    "explanation": "문제를 풀고 나서 표시되는 정답 해설 문장"
  }
];
```

## 2. 타 과목 예시 스키마 코드

### 🔬 과학(Science) 과목 적용 예시

```javascript
export const chapters = [
  {
    "id": "1",
    "title": "Ⅰ. 화학 반응의 규칙성",
    "sections": [
      {
        "id": "1-1",
        "title": "1. 물질의 변화와 화학 반응식",
        "subsections": [
          {
            "id": "1-1-1",
            "title": "01 물리 변화와 화학 변화",
            "objective": "물질의 고유한 성질은 변하지 않고 모양이나 상태가 변하는 물리 변화와 성질이 다른 새로운 물질로 변하는 화학 변화의 차이를 분자 배열의 변화로 설명할 수 있다.",
            "page": 12,
            "fillInTheBlanks": [
              {
                "sentence": "물질의 성질은 변하지 않으면서 모양이나 상태가 변하는 현상을 [물리 변화]라고 한다.",
                "answer": "물리 변화"
              },
              {
                "sentence": "철이 녹슬거나 양초가 타는 현상처럼 원래 물질과 성질이 전혀 다른 새로운 물질로 변하는 현상을 [화학 변화]라고 한다.",
                "answer": "화학 변화"
              }
            ],
            "flashcards": [
              {
                "term": "물리 변화",
                "definition": "모양이나 상태만 바뀔 뿐 물질을 구성하는 분자 자체는 변하지 않아 물질 고유의 성질이 그대로 유지되는 현상."
              },
              {

              }
            ]
          }
        ]
      }
    ],
    "subjectiveQuizzes": [
      {
        "id": "subq_1_1",
        "question": "물이 끓어 수증기가 되는 현상과 물을 전기 분해하여 수소와 산소를 얻는 현상을 물리 변화와 화학 변화 관점에서 원자 및 분자의 배열 변화를 포함하여 비교 서술하시오.",
        "expectedAnswer": "물이 끓어 수증기가 되는 것은 분자의 종류와 원자 배열은 변하지 않고 분자 사이의 거리만 멀어지는 물리 변화인 반면, 물을 전기 분해하는 것은 물 분자가 분해되어 원자 배열이 재조합되고 수소 분자와 산소 분자라는 새로운 물질이 형성되는 화학 변화이다."
      }
    ]
  }
];
```

## 3. 데이터 마이그레이션 시 주의 사항
- **고유 ID**: 모든 `id`값은 다른 챕터와 중복되지 않도록 설정해야 합니다. (예: `1-1-1`, `2-1-1` 등 트리형 규칙 권장)
- **JSON 표기 규칙**: 자바스크립트 내부에 내보낼(export) 때 데이터 규격을 만족하려면 큰따옴표(`"`)를 엄격하게 사용하는 표준 JSON 스타일을 유지하는 것이 좋습니다.
- **줄바꿈 처리**: 여러 문장을 개행할 때는 따옴표 내부에서 문자열 리터럴 개행을 쓰거나 `\n` 이스케이프 코드를 활용하십시오.
